<p>I am looking to mimic spreadsheet functionality in a content section of Drupal. Let me know if anyone knows a resource. Also, can we explore adding an editor?</p><p> </p><p><a href="http://tinymce.moxiecode.com/index.php" target="_blank">http://tinymce.moxiecode.com/index.php</a></p><p> </p><h1>TinyMCE - Javascript WYSIWYG Editor</h1>

					<p>TinyMCE is a platform independent web based Javascript HTML WYSIWYG editor control released as Open Source under <a href="http://tinymce.moxiecode.com/license.php">LGPL</a> by <a href="http://www.moxiecode.com/">Moxiecode Systems AB</a>.
It has the ability to convert HTML TEXTAREA fields or other HTML
elements to editor instances. TinyMCE is very easy to integrate into
other Content Management Systems. </p>
