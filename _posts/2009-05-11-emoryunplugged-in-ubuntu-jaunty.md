<p>Not much for a first post, but at least it’s content.

Emory has two more-or-less campus-wide wireless networks: EmoryGuest and EmoryUnplugged, preferring the latter for University users with Emory network IDs. IT gives <a href="http://it.emory.edu/showdoc.cfm?docid=5232">directions</a> for connecting to EmoryUnplugged with several flavors of Windows and Mac. It does not, unfortunately, extend these directions to any flavors of Linux.

Just today Emory IT got my network ID set up. My laptop happens to be running Ubuntu — Jaunty in particular. While IT doesn’t officially support my OS, their Mac directions gave me the hints I needed to connect. The only configuration change I needed to make was to set the Authentication to PEAP. Everything else kept its default settings (WPA/WPA2 for Security, and MSCHAPv2 for Inner Authentication). That done, and my Emory NetID entered, it works like a charm.

So far I’m pretty happy with Ubuntu Jaunty.</p>
