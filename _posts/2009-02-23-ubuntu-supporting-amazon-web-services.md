Would you rather build a cloud on your<br />own infrastructure to get a feel for things while the public clouds "cure"?<br />Starting with Karmic's predecessor, Jaunty Jackalope (soon to go to code<br />freeze), UC Santa Barbara's open-source cloud project, Eucalyptus, will be<br />included in every install package.</p>
