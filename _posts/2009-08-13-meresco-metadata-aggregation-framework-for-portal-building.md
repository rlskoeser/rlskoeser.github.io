<p>I stumbled across this (MERESCO) on a listserv.&nbsp; It is similar to what Southcomb is designed to do, in that it is used to aggregate OAI-PMH, RSS and Web site metadata into a single portal, which has social networking functions built on top.</p><p>The tools are all open source, and for the most part based in Python</p><p>http://meresco.com/page?id=introduction</p><p>
  MERESCO is a next generation library of components for (meta)data management.
</p>

<p> It supports creating flexible, scalable and open (meta)data
management systems such as repositories, search engines and archives.
</p><p>It has been developed in the Netherlands by SURF-foundation, SURF-net,<br />KennisNet and CQ2. It actually is a larger library of components for<br />metadata management (including data normalization, integrated and<br />facetted search etc), but one of the earliest set of components<br />contains a complete OAI harvester with Web UI.<br /><br />For general information see http://meresco.com<br /><br />For the harvester in particular see<br />http://meresco.com/page/package_information?package=meresco-harvester<br /><br />It is all Open Source and actively developed.&nbsp; SURF-foundation<br />organizes a conference on MERESCO September 29th 2009.<br /><br /></p>
