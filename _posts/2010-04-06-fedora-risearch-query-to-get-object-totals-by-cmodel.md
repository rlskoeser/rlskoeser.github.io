<p>I&nbsp; just figured out how to do a count query in iTQL (one of the triple query languages for the triplestore that we're using with Fedora) and want to document it so I don't lose it.&nbsp; This returns a list of content models and the number of active objects for each cmodel.</p><pre>select $cmodel<br />count(select $item from&nbsp; &lt;#ri&gt;<br />where $item &lt;info:fedora/fedora-system:def/model#hasModel&gt; $cmodel<br />and $item &lt;info:fedora/fedora-system:def/model#state&gt; &lt;fedora-model:Active&gt;)<br />from &lt;#ri&gt;<br />where $item &lt;info:fedora/fedora-system:def/model#hasModel&gt; $cmodel<br />having $k0 &lt;http://mulgara.org/mulgara#occursMoreThan&gt; '0.0'^^&lt;http://www.w3.org/2001/XMLSchema#double&gt; ;<br /><br /></pre><p>Current totals for Fedora staging (where we have been testing the LSDI ingest -
