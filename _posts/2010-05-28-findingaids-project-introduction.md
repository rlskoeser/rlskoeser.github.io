<p>FindingAids is a web application that stores and manages records of manuscripts &amp; archives. </p><p><strong>Users</strong>: Researchers, Scholars or any Internet user </p><p><strong>Administrators</strong>: Emory Library colleagues</p><p><strong>Data</strong></p><p>Each such manuscript &amp; archive record is represented using an XML file following the <a href="http://www.loc.gov/ead">EAD schema</a>. The XML files are provided by the administrators at Emory library. They are stored in eXist XML database. There's <a href="http://kamina:8080/exist/sandbox/sandbox.xql">a web interface</a> to query the xml files using XQuery. An example query is <strong>collection('/db/FindingAids/emory')//archdesc/did/origination/famname</strong></p><p><strong>Source code:</strong></p><p>SVN https://svn.library.emory.edu/svn/findingaids</p><p>a "svn checkout" will get you a copy of findingaids source code as well as a copy of eulcore which is shared by several projects in the libarary. Findingaids depends one eulcore. Note that, changes to the eulcore has to be done under a eulcore trunk which is a seperate copy from everything checked out for the finding aids project. </p><p><strong>Technologies:</strong></p><p>Python, Django</p><p>XML, XQuery, XPath, eXist</p><p><a href="http://sphinx.pocoo.org/index.html">Sphinx </a>(also <a href="http://docutils.sourceforge.net/">DocUtils&nbsp; </a>which Sphinx is based on)</p><p>&nbsp;</p><p><strong>Installation:</strong></p><div id="edit-files-66-description-wrapper" class="form-item">
</div><p>To make the installation (and life) easier, download <a href="http://techknowhow.library.emory.edu/sites/techknowhow.library.emory.edu/files/mysetup.txt">this file</a> and run it as a shell script. </p><p>&nbsp;</p><p><strong>Settings (under Linux)</strong></p><p>1) copy localsettings-sample.py to localsettings.py</p><p>&nbsp;&nbsp; Changes, customized by yourself, need to made to localsettings.py: </p><p><strong>EXISTDB_SERVER_URL&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; = 'http://findingaids:findingaids@kamina.library.emory.edu:8080/exist/xmlrpc'<br />EXISTDB_ROOT_COLLECTION = "/FindingAids/emory"</strong><strong>&nbsp;</strong></p><p><strong>EXISTDB_PREVIEW_COLLECTION = "preview/fa-jing"<br />EXISTDB_TEST_COLLECTION = "/test/fa-jing"</strong></p><p><strong>SECRET_KEY = 'jinganrstafpfpluxcv'</strong></p><p><strong>changes for LDAPS to localsettings.py</strong></p><p><strong><br /></strong></p><p>2) Create database tables so that the adiministration functions are available to you. Run for findingaids website:</p><p><strong>python manage.py syncdb</strong> </p><p>A database will be created for your development work. Create a username and password for the databse administrator. Use them to login for administration functions. </p><p>&nbsp;</p><p>&nbsp;</p>
