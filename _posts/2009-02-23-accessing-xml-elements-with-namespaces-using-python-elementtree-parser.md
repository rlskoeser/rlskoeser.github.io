<p>In the following XML, the record element has an associated namespace.<br />If there was no namespace, the elements could be accessed like this:</p><p> </p><pre>from xml.etree import ElementTree<br />etree=ElementTree.fromstring(xml)<br />record=etree.find("source_locate_response/source_customize_info/record")<br />datafield=etree.find("source_locate_response/source_customize_info/record/datafield")<br />subfield=etree.find("source_locate_response/source_customize_info/record/datafield/subfield")</pre><p> </p><p> </p><p>Since these elements have a namespace, the parser has to refer to the namespace when it accesses the elements:</p><p> </p><pre>from xml.etree import ElementTree<br />etree=ElementTree.fromstring(xml)<br /><strong>ns='{http://www.loc.gov/MARC21/slim}'</strong><br />record=etree.find("source_locate_response/source_customize_info/<strong>%</strong>srecord" % (ns))<br
