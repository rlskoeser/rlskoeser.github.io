<p>This is an example taken from the ETD project of how to use a cached RSS feed.&nbsp; The advantage of caching is that it will not retreive that external feed every time the feed is requested, but it will retreive a new version after the cache has expired. This saves on bandwidth and makes the site more effecent.<br /><br /></p><p>Setup the config for the cache and RSS feed in config.xml. In this example I only need two variables.<br />url - The URL of the RSS Feed<br />lifetime - Expiration time of cache (in seconds)</p><p>This will expire the feed after 1 hour</p><div style="background-color: SkyBlue;">&lt;news_feed&gt;<br />&nbsp;&nbsp;&nbsp;&nbsp; &lt;url&gt;https://digital.library.emory.edu/taxonomy/term/22/all/feed&lt;/url&gt;<br />&nbsp;&nbsp;&nbsp;&nbsp; &lt;lifetime&gt;3600&lt;/lifetime&gt;<br />&lt;/news_feed&gt;</div><p>&nbsp;</p><p>Create a function to return a cache object:</p><div style="background-color:SkyBlue">public function createCache($lifetime){<br /><br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; //refresh time of cache<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; //make sure value is null if value is not set or empty - null value means forever<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; $lifetime =&nbsp; (empty($lifetime) ? null : $lifetime);<br /><br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; $frontendOptions = array('lifetime' =&gt; $lifetime, 'automatic_serialization' =&gt; true);<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; $backendOptions = array('cache_dir' =&gt; '/tmp/', "file_name_prefix" =&gt; "ETD_news_cache");<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; $cache = Zend_Cache::factory('Output', 'File', $frontendOptions, $backendOptions);<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; return $cache;<br />&nbsp; }</div><p>&nbsp;</p><p>Create a function to read the feed this is also where the cache function is used:</p><div style="background-color:SkyBlue">&nbsp; public function getNews(Zend_Config $config) {<br />&nbsp;&nbsp;&nbsp; // ETD news - rss feed from drupal site<br />&nbsp;&nbsp;&nbsp; if (! isset($config-&gt;news_feed-&gt;url)) {<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; throw new Exception("News feed is not configured");<br />&nbsp;&nbsp;&nbsp; }<br /><br />&nbsp;&nbsp;&nbsp; try {<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; $news_feed = $config-&gt;news_feed-&gt;url;<br /><br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; //Set Feed_Reeder to use cache<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; $cache = $this-&gt;createCache($config-&gt;news_feed-&gt;lifetime);<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Zend_Feed_Reader::setCache($cache);<br /><br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; //Read the feed<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; $news = Zend_Feed_Reader::import($news_feed);<br />&nbsp;&nbsp;&nbsp; } catch (Exception $e) {<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; throw new Exception("Could not parse ETD news feed '$news_feed' - " . $e-&gt;getMessage());<br />&nbsp;&nbsp;&nbsp; }<br /><br />&nbsp;&nbsp;&nbsp; return $news;<br />&nbsp; }</div><p>
In the action call the above function like this to pass it to the view:</p><div style="background-color:SkyBlue">$this-&gt;view-&gt;news = $this-&gt;getNews($config);</div>
<p>In the view access the elements of the feed like this:</p><div style="background-color:SkyBlue">&lt;? foreach ($this-&gt;news as $item): ?&gt;<br />&nbsp; &lt;div&gt;<br />&nbsp;&nbsp;&nbsp; &lt;h3&gt;&lt;?= $item-&gt;getTitle() ?&gt;&lt;/h3&gt;<br />&nbsp;&nbsp;&nbsp; &lt;?= $item-&gt;getDescription() ?&gt;<br />&nbsp; &lt;/div&gt;<br />&lt;? endforeach ?&gt;</div>
