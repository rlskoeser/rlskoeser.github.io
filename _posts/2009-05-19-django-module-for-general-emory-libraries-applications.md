<p>I’ve created a new subversion project called <a href="https://svn.library.emory.edu/svn/django-eulcore/"><code>django-eulcore</code></a> as a library for reusable Django code inside the Library. Currently it contains modules for SOAP views and testing, as well as functionality for tying LDAP authentication into Django apps. So far all of the code is based on stuff already out in the blogosphere, but Rebecca and I have hacked the to improve functionality and reusability. Look for documentation here on techknowhow as soon as I figure out where those docs belong.</p>

<p>Apologies to any readers from outside Emory Libraries: Repository access is limited to our developers for now, though I hope to be able to open-source as much as possible in time.</p>
