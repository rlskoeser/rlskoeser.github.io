<p>A new version of the ETD software was deployed, with support for undergraduate Honors Thesis.  This year, electronic submission for the Honors students will be voluntary.  This update includes adapting all management workflow and functionality so it will work for the Graduate School admin <em>and</em> the College Honors Program admin, and also modifying the edit pages and require fields for submission based on the type of student (Honors ETDs are not sent to ProQuest, so certain data we collect for grad students is not needed).  Also, because the Honors Programs change from time to time, the logic for the Programs hierarchy used in the ETD site has been revamped to work better with Graduate and Honors programs, and to allow the Honors admin to edit the programs as needed.</p><p>This update also updates the committee editing logic to allow for including former Emory faculty as committee chair or committee member, a configuration test script to run a sanity-check on all site configuration files, and other minor fixes.</p>
