<p>https://fedora-commons.org/confluence/display/AKUBRA/Akubra+Project</p><p>The Akubra Project is a new effort to provide a pluggable file storage
interface that can be adapted to almost any storage subsystem. Akubra
will support both ordinary and transactional storage systems, but makes
simplifying assumptions about "Blobs" in order to achieve a high level
of interoperability. Akubra is part of the <a href="https://fedora-commons.org/confluence/display/DEV/Storage+Delegation+Project" title="Storage Delegation Project">Storage Delegation Project</a>.  </p>
