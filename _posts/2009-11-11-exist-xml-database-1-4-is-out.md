<p>Exist XML database version 1.4 is now in stable release.&nbsp; My plan is to install it on Kamina, for the development environment, and convert the databases from Wilson.&nbsp; This will give the Exist databases it's own seperate instance of Tomcat/Jetty as well as giving the new functionality of 1.4.&nbsp; Of special interest with the new version is it's integration with Lucene. </p><p>Exist 1.4 has been installed on Kamina.&nbsp; Since I did a restore from the Wilson databases, the kamina install should have all your user IDs.&nbsp; No index work has been done.&nbsp; I used this link (http://exist-db.org/lucene.html) to learn more about Exist with Lucene&nbsp; Please try it out and give any feedback.</p>
