<p><a href="http://thorkelin.library.emory.edu/~croddy/screenshots/sqliteman.png" target="_blank"><img src="http://thorkelin.library.emory.edu/~croddy/screenshots/sqliteman.png" alt="Sqliteman screenshot" title="Sqliteman screenshot" width="160" height="116" style="float: right; margin: 8px; margin-top: 0px; margin-right: 0px;" /></a>Anyone who has been looking for a more convenient way to manage SQLite databases besides the <span class="tt">sqlite</span> command-line interface might want to check out <a href="http://sqliteman.com/" target="_blank">Sqliteman</a>. It is a GUI tool for Windows and Linux, written using QT4, that will probably make life much easier.</p><!--break--><p>Since the development builds of <a href="http://code.google.com/chromium/" target="_blank" title="Chromium">Chromium</a>&nbsp;that I have been using still have no configuration interface, but support quite a lot of the Google Chrome configuration system, I've had to spend a lot of time manually modifying the SQLite files that Chromium uses for storing this stuff. After a short time using Sqliteman I have been able to get a much clearer understanding of the config databases and edit some of my keyword searches much more quickly.</p><p>For Windows, Sqliteman ships a typical installer binary. I ran into some trouble (i.e., impatience) building it on Ubuntu, but <a href="https://launchpad.net/~emerzh/+archive/ppa" target="_blank">Brice Maron's personal package archive</a> at launchpad.net permits installing Sqliteman using APT.</p>
